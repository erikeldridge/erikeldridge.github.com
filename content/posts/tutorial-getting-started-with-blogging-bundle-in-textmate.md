motivation:
I just discovered the blogging bundle in textmate. It works great, but I was a bit confused about how to get started.

references:

- [http://blog.macromates.com/2006/blogging-from-textmate/](http://blog.macromates.com/2006/blogging-from-textmate/)

prerequisites:

- you need to have a Wordpress blog (or any blog that uses the MetaWeblog API)

steps:

- launch textmate
- from the _Bundles_ menu, select _Blogging_![](http://img.skitch.com/20090806-g2pndwgfbqa8pyxd5nnkgt9k7c.preview.jpg)
- select _Setup Blogs_ from menu
  ![](http://img.skitch.com/20090806-rux5kc8p5hy34tcb7c2m7rrjsn.preview.jpg)
- in the file that opens, enter your blog’s name and the url for it’s [xmlrpc endpoint](http://faq.wordpress.com/2006/01/20/xml-rpc-and-desktop-apps/)![](http://img.skitch.com/20090806-jbarfcf2ppmejptseba8q88wi6.preview.jpg)![](http://img.skitch.com/20090806-npay3at17s2tyx1qeygx6riumg.preview.jpg)
- save and close that file
- create a new file, and, optionally, put some text in it
- go back into the _Bundles_—\> _Blogging_ menu and select _Post To Blog_![](http://img.skitch.com/20090806-bmt39g55m5a8cqn357cuybkmtf.preview.jpg)
- when prompted, enter a title for your blog post, and then your Wordpress username and password (3 separate prompts).
  ![](http://img.skitch.com/20090806-ppq4xfm78iribibgka4kftqy5t.preview.jpg)
  The bundle will then publish the contents of your file as a post to your blog.
