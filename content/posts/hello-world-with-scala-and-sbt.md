**Preamble**

I'd like to learn more about [Scala](http://www.scala-lang.org/ "Scala's Web site").  To get started, I'm going to run through a hello world example on [sbt's Google Code site](http://code.google.com/p/simple-build-tool/ "sbt's Google Code site").  Here are my system details:

- Mac 10.6.6
- Scala 2.7.7
- sbt 0.7.4
- Java 1.6

**Steps**

1. Create a new sbt project by simply running the command _sbt_.  I set the name to "hw", the organization to "foo.com", and the project version to "1.0", and accepted the defaults for the Scala and sbt versions but just hitting enter.  After building the project, sbt will leave a command prompt open.  Close this by typing ctrl-D.  Note: The [quick start](http://code.google.com/p/simple-build-tool/wiki/Setup#Quick_Setup "the sbt documentation's quick start") creates the code first, but then doesn't talk about where to put it, which I found a bit confusing, so I'm presenting it here in reverse order.

   \[caption id="attachment\_1139" align="alignnone" width="552" caption="Screen shot of sbt creating a project"\]![Screen shot of sbt creating a project](/wp-content/uploads/2011/02/screen-shot-2011-02-23-at-12-34-47-am.png)\[/caption\]
1. Create the file _hw.scala_ in the _src/main/scala/_ directory generated by sbt:
   _echo 'object Hi { def main(args: Array\[String\]) { println("Hi!") } }' > src/main/scala/hw.scala_
1. Compile the project: _sbt compile_

   \[caption id="attachment\_1143" align="alignnone" width="552" caption="Screen shot of compiling a project"\] [![Screen shot of compiling a project](/wp-content/uploads/2011/02/scrnshot.png)](/wp-content/uploads/2011/02/scrnshot.png)\[/caption\]
1. Run the project: _sbt run_

   \[caption id="attachment\_1142" align="alignnone" width="635" caption="Screen shot of running a project"\]![Screen shot of running a project](/wp-content/uploads/2011/02/screen-shot-2011-02-23-at-12-36-42-am.png)\[/caption\]

**Conclusion**

So that's that. It's just hello world, but I found it helpful as an introduction to sbt's relationship to Scala.  Time to call it a day, like these birds are doing:
