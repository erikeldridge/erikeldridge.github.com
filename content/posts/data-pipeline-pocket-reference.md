The [_Data Pipeline Pocket Reference_ by James Densmore](https://www.oreilly.com/library/view/data-pipelines-pocket/9781492087823/) is a practical overview of pipeline concepts and terminology. It demonstrates most concepts using framework-agnostic Python scripts. It also provides a good introduction to MLOps by recommending popular solutions to common problems, like Apache Airflow for orchestration. I'd recommend it to anyone ramping up on MLOps.
